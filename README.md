# AI 리터러시 강사

## 바로 사용하기

👉 **[강사 프로필 열기](https://hiation33-arch.github.io/instructor-profile/landing.html)**

---

AI 리터러시 강사 은미의 소개 페이지입니다.

## 구성

| 파일 | 설명 |
|------|------|
| `landing.html` | 인트로 랜딩 페이지 (커튼 배경 + BGM) |
| `index.html` | 강사 프로필 본문 페이지 |
| `curtain.png` | 랜딩 배경 이미지 |
| `Morning_Breeze.mp3` | 배경 음악 |
| `images/` | 이미지 리소스 폴더 |

## 사용 방법

`landing.html`을 브라우저에서 열면 시작됩니다.  
화면을 클릭하면 음악이 재생되며 프로필 페이지로 이동합니다.
